# 🖥️ Eleven Meme Generator

A futuristic cyberpunk meme creator built with HTML5 Canvas, JavaScript, and modern CSS.

Create high-quality memes instantly using custom templates or your own uploaded images — all inside a sleek neon-themed interface inspired by retro cyberpunk aesthetics.

---

## 🚀 Live Demo

🌐 https://hitesharya11.github.io/eleven-meme-generator/

---

## ✨ Features

- 🎨 Cyberpunk neon UI design
- 🖼️ Upload custom images
- 📂 Drag & drop image support
- ⚡ Real-time meme preview
- ✍️ Top & bottom caption editing
- 🔠 Adjustable font size controls
- 🎯 Custom text colors & outline colors
- 🧠 Automatic text wrapping
- 🖨️ High-quality PNG export
- 🌌 Built-in CORS-free SVG templates
- 📱 Responsive modern layout

---

## 🖼️ Built-in Templates

The app includes 4 built-in SVG meme templates:

- 🌅 Cyber Sunset
- 🌃 Neon City
- 🎮 Retro Gamer
- 🪐 Space Saturn

All templates are locally bundled to avoid CORS issues when exporting canvas images.
