# 🚀 Run Locally

Because the application loads local SVG templates into the HTML5 Canvas, opening the project directly using:

```text
file:///
```

may trigger browser security restrictions.

Run the project using a local web server instead.

---

## Using Python

### Step 1
Open terminal inside the project folder:

```bash
cd eleven-meme-generator
```

### Step 2
Run local server:

```bash
python -m http.server 8000
```

### Step 3
Open browser:

```text
http://localhost:8000
```

---

## Using Node.js

### Step 1
Run temporary local server:

```bash
npx http-server
```

### Step 2
Open the URL shown in the terminal.

Usually:

```text
http://127.0.0.1:8080
```
